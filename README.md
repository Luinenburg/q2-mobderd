# Quower Quefence
Welcome to Quower Quefence! A tower defence game built on quake

Defend yourself from hoards of enemies trying to attack you

Build towers to help you

Upgrade them as the danger levels increase

Press V to check the help screen

Go to releases and download the latest release. After unzipping, drop your baseq2 folder in that folder.

Run quower_quefence.sh to start the game
